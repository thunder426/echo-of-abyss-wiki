Echo of Abyss

A single-player roguelike deckbuilder. Command the forces of the Abyss Order, summon demons, weave runes, and conquer the void.

About the Game

Echo of Abyss is a single-player roguelike deckbuilder where you take the role of a hero fighting through 15 increasingly difficult encounters across 4 acts. Between fights, you earn new cards, relics, and talents to build an ever-more-powerful deck.


Core Loop: Start a run with a starter deck → Fight encounters → Earn rewards (cards, relics, talent points) → Grow your resources → Defeat the final boss to unlock permanent reward cards.

Key Pillars

Tactical Board Play

5 minion slots per side. Place minions, cast spells, set traps, and deploy environments. Every board position matters.

Dual-Resource System

Essence summons minions. Mana fuels spells, traps, and runes. Grow one each turn — combined cap of 11.

Deep Synergy

Void Imp swarms, Corruption debuffs, Rune auras, Ritual combos, and Void Bolt scaling create layered strategies.

Roguelike Progression

Each run offers different card rewards, relic choices, and talent paths. Permanent unlocks carry across runs.
