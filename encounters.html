Passives

  • Feral Reinforcement: First Human summoned each turn → enemy draws a random Feral Imp. Once per turn.
  • Void Unraveling:
    • Human summoned → spawn a 100/100 Void Spark on enemy board
    • Feral Imp summoned → consume 1 Void Spark, grant +100/+100 to that imp
    • End of enemy turn → corrupt 1 random Void Spark and transfer it to player board (clogs player slots)

Champion: Corrupted Handler

ATK 300HP 300
Summoned after 3 void sparks created. Aura: Whenever a Void Spark is summoned, deal 200 damage to player hero.

Strategy

The Void Unraveling loop is the core threat: Humans spawn Void Sparks, Imps consume them for buffs, and unclaimed Sparks get transferred to your board. The champion makes every new Spark deal 200 hero damage. Kill Void Spawners and clear Sparks before they pile up.

Act 3 The Void Rift

Fight 7: Rift Stalker
HP: 3800
“The Void Rift World — a place where reality has frayed. Rift Stalkers phase between dimensions, attacking from angles that shouldn't exist.”

Key Cards

  • Phase Stalker (2E + 1 Spark): 400/300, Swift — fast, aggressive Spirit
  • Void Behemoth (3E + 2 Sparks): 400/600, Guard — massive wall
  • Void Bolt (3M): 500 Void Bolt damage to hero
  • Void Pulse (1M + 1 Spark): Draw 3 cards

Passives

  • Void Rift: Generates Void Sparks at the start of each enemy turn.
  • Void Empowerment: Buffs minion stats based on available Void Spark count.
Fight 8: Void Aberration
HP: 4600
“A Void Aberration — a creature that should not exist in any plane. It was assembled from the broken remnants of things consumed by the rift.”

Key Cards

  • Dimensional Breach (1M + 2 Sparks): Summon 3 Void Sparks — creates Spark loops
  • Rift Collapse (2M + 1 Spark): 200 AoE damage to all enemy minions
  • Same Void Rift toolkit as Fight 7, but with more Spark generation

Passives

  • Void Rift: Turn start Void Spark generation.
  • Void Detonation: Void Sparks can detonate for damage when threshold is reached.
BOSS Fight 9: Void Herald
HP: 6500
“The Void Herald speaks with the voice of the Abyss itself. It has crossed countless worlds before this one. It carries a message: the Abyss Sovereign is coming, and nothing will remain.”

Key Cards

  • Void Rift Lord (4E + 3 Sparks): 400/600, On play: set enemy Mana to 0 next turn — devastating mana denial
  • Full Void Rift toolkit: Phase Stalkers, Void Behemoths, Dimensional Breach, Rift Collapse

Passives

  • Void Rift: Turn start Void Spark generation.
  • Void Mastery: Halves all Void Spark costs — massive resource advantage. The Herald plays Spark cards at half price.

Strategy

Extremely high HP (6500). Void Mastery halves all Spark costs, giving the Herald a massive tempo advantage. Void Rift Lord can lock down your Mana entirely. You need aggressive tempo and resource denial to keep up.

The ultimate endurance test. 10000 HP with three stacking passives — Void Spirits are buffed by Void Might, spells hit harder from Void Precision and Dark Channeling.
